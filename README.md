# 📸 Gallery App

## 🚀 Features
- Register / Login (JWT)
- Upload images
- Like photos ❤️
- Search photos
- Multiple upload

## 🛠 Tech Stack
- FastAPI (Backend)
- ReactJS (Frontend)
- SQLite (Database)

## ▶️ Run project

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
